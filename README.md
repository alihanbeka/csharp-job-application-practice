# C# Job Application Practice

This is a beginner-level C# console application created to practice core programming concepts through a simple job application tracking system.

## Features

- Add a job application
- List all job applications
- Change application status
- Delete an application
- Save applications to a text file
- Load saved applications when the program starts
- Validate user input when selecting an application

## Practiced Concepts

- Classes and objects
- Constructors
- Constructor overloading
- List<T>
- foreach loops
- for loops
- while loops
- if / else conditions
- Methods
- File reading and writing
- Basic CRUD logic
- Input validation with int.TryParse

## Learning Purpose

This project is part of my learning path toward backend and full-stack development.  
The goal is to understand the basics of C#, CRUD logic, data persistence, and clean method-based code structure before moving into .NET Web API and databases.
